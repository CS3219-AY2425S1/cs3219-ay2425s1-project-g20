## Getting Started

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:CS3219-AY2425S1/cs3219-ay2425s1-project-g20.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the local development server
   ```sh
   npm run dev
   ```
